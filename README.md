# Pogchamp Online

## Installation

Install node.js, npm, then clone this repository

```sh
$ git clone https://github.com/Brenterino/pogchamp-online pogchamp-online
```

Install the node dependencies

```sh
$ cd pogchamp-online
$ npm install
```

Start the server

```sh
$ npm run server
```

Run the client

```sh
$ npm run client
```

View in browser at `http://localhost:8081/`
